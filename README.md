# 🦈 SharkList

Um catálogo digital dedicado às espécies de tubarões do mundo, reunindo informações científicas, imagens de alta qualidade e uma experiência moderna de navegação.

> **Status:** Em desenvolvimento 🚧

## Objetivo

O projeto tem como objetivo criar um dos catálogos online mais completos sobre tubarões, apresentando informações de forma organizada, acessível e visualmente atrativa para estudantes, pesquisadores, mergulhadores e entusiastas da vida marinha.

## Funcionalidades

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/sharks.git
```

2. Acesse a pasta do projeto:

```bash
cd sharklist
```

3. Abra o arquivo `index.html` em um navegador ou utilize uma extensão como **Live Server** no Visual Studio Code.
